Welcome to the documentation of ttta: Tools for Temporal Text Analysis!
===================================

ttta is a Python library for all researchers interested in working with diachronic text data.
That is, the modules in this library aid the user by providing NLP models that specifically deal with changing language in text corpora.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
